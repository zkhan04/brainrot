# brainrot

brainrot is a Chrome extension that transforms your tab into a brainrot wonderland with "brainrot" points. Earn points by interacting with web elements and spend them in the shop to trigger chaotic effects. This was a submission for the jia.seed Brainrot Hackathon. 

**Features**:
- Point system: earn brainrot points by hovering over buttons and interacting with pages.
- Brainrot shop: a dedicated UI to purchase and activate effects.
- Effects: sounds like "Hawk Tuah" and "I just lost by dawg", visuals like Subway Surfers gameplay overlays, and DOM manipulation that replaces standard webpage text with brainrot terminology.

**Tools Used**
- Languages: HTML, CSS, Javascript
- APIs: Chrome Extension API

**How To Use**
- Open Chrome and navigate to ```chrome://extensions```
- Enable Developer Mode via the top-right toggle
- Click "Load unpacked" and select the brainrot folder containing ```manifest.json```
- The extension is installed!
